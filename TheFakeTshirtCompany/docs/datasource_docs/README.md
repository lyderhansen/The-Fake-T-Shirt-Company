## Correlation Fields

All sources can be correlated using these key fields:

| Field | Description | Used In |
|-------|-------------|---------|
| `demo_id` | Scenario identifier | All sources |
| `demo_host` | Server/device name | Windows, Linux, Perfmon |
| `user` / `userPrincipalName` | User identity | Entra ID, WinEventLog, Exchange |
| `src` / `dst` | Network addresses | ASA, Meraki |
| `order_id` | Order identifier | Orders, ServiceBus, Access |
| `meeting_id` | Meeting identifier | Webex sources |
| `timestamp` | Event time | All sources |

## Quick Filters

```spl
# All events from a scenario
index=fake_tshrt demo_id=exfil

# All events from a host
index=fake_tshrt demo_host="SQL-PROD-01"

# All events for a user
index=fake_tshrt (user="alex.miller" OR userPrincipalName="alex.miller@theFakeTshirtCompany.com")

# Cross-source timeline
index=fake_tshrt demo_id=exfil | sort _time | table _time, sourcetype, host, message
```
